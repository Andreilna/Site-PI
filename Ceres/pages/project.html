Greenrise

Sistema de Fazenda Vertical com Inteligência Artificial e Automação Completa

Greenrise é a solução completa para quem quer implementar fazendas verticais inteligentes. Com sensores avançados, controle automatizado e IA que aprende continuamente.

Características Principais:
  • ✓ 7 componentes integrados para automação total
  • ✓ Sensores de temperatura, umidade, pH, CO₂ e nutrientes
  • ✓ Rede neural treinada para decisões autônomas
  • ✓ Controle de iluminação LED, irrigação e nutrição
  • ✓ Banco de dados em tempo real para análises
Resultados Esperados:
90% redução de água
80% menos pesticidas
3x mais colheitas/ano

ODS 2 • ODS 9 • ODS 11 • ODS 12

Selene

Sistema Inteligente de Monitoramento de Cultivo de Cogumelos com Deep Learning e IoT

Selene é uma plataforma inteligente desenvolvida para o monitoramento do cultivo de cogumelos, integrando sensores ambientais, câmeras e inteligência artificial. O sistema realiza o acompanhamento em tempo real de variáveis como temperatura, umidade e possíveis focos de contaminação, auxiliando produtores na prevenção de infecções e na tomada de decisões rápidas e precisas.

Características Principais:
  • ✓ Monitoramento em tempo real de temperatura e umidade
  • ✓ Integração com sensores IoT e câmeras de alta resolução
  • ✓ Detecção de anomalias visuais usando Deep Learning
  • ✓ Alertas automáticos de possíveis contaminações
  • ✓ Dashboard intuitivo para acompanhamento remoto
  • ✓ Aplicação mobile e desktop integradas via API REST
Funcionalidades:
Monitoramento em Tempo Real Detecção de Parasitas Visão Computacional Alertas Inteligentes
Impacto na Produção:
24/7 monitoramento contínuo
60% menos perdas
IA suporte automatizado
Alinhamento ODS:

ODS 2 • ODS 9 • ODS 12
